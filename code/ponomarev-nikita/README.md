# Конвертор площадей

Выполнил:

 - Пономарев Никита
 - ННГУ, Институт ИТММ, каф. МОСТ, группа 381606м1

## Задание

Реализовать класс конвертор площадей с поддержкой следующих переводов:

 - Квадратный метр <-> Квадратный километр
 - Квадратный метр <-> Гектар 
 - Квадратный метр <-> Ар(сотка)
 - Квадратный метр <-> Квадратный сантиметр
 - Квадратный метр <-> Квадратный милиметр
 - Квадратный метр <-> Квадратный ярд
 - Квадратный метр <-> Квадратный фут
 - Квадратный метр <-> Квадратный дюйм
 - Квадратный метр <-> Квадратный акр
 - Квадратный метр <-> Квадратная миля

Соотношения единиц измерения площади можно найти на сайте www.fxyz.ru в разделе ["Единицы измерения площади"][square]
